# Brew & Co. – Coffee Shop Website Demo

![Brew & Co Screenshot](images/screenshot.png)

## Overview
This is a **responsive, 3-page coffee shop website** built with **HTML, CSS, and JavaScript**.  
It’s designed to showcase a small coffee business, with a modern and clean layout.

## Pages
1. **Home** – Hero section, featured menu items, welcoming message  
2. **Menu** – Drinks and snacks, responsive menu cards  
3. **Contact** – Contact form, map embed, social links  

## Features
- Fully **responsive design** (desktop & mobile)  
- Smooth **hover effects & animations**  
- Easy to **customize and host**  
- Ready for **custom domain integration**  
- **Clean, commented code** for easy editing  

## Technologies
- HTML5  
- CSS3 (Flexbox, Grid)  
- JavaScript (for animations & interactivity)  
- Hosted on **GitHub Pages**  

## Live Demo
[Check it live here](https://thnav.github.io/Brew-Co/)

## Optional Extras for Clients
- Custom logo design (Affinity Designer)  
- Custom domain setup  
- Contact form integration  

## How to Use
1. Clone the repository  
2. Open `index.html` in your browser  
3. Modify menu items, colors, and text to your brand  
4. Upload to any static hosting platform (Netlify, Vercel, GitHub Pages)
